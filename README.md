# tweetb
Simple Twitter bot using Selenium which likes tweets and retweets.


## Usage

### Install Selenium using
```pip install selenium```

### Replace the placeholder username and password with credentials of the desired account

### Run the bot using
```python3 app.py```
#### or
```python app.py```
